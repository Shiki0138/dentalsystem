### 2. 操作フロー

#### 受付スタッフの典型的な業務フロー
1. **来院患者の確認**
   - ダッシュボードで本日の予約確認
   - 患者名クリックで詳細表示

2. **新規予約登録**
   - クイックアクション「新規予約」
   - 患者検索 → 日時選択 → 確定

3. **診察完了処理**
   - 予約ステータスを「診察済」に更新
   - 次回予約の登録
